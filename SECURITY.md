# Security Policy

## Educational Scope

The scripts and guides contained in this repository are developed strictly for academic, research, and educational purposes. They are intended for use only in isolated lab environments (virtual machines) where the user has full administrative control.

## Reporting a Vulnerability

If you discover a security vulnerability within this project itself, please do not disclose it publicly.

Instead, open an issue labeled `security` or contact the repository owner directly. Given the educational nature of the project, reasonable effort will be made to remediate architectural or dependency vulnerabilities.

## Responsible Use

By cloning or downloading materials from this repository, you agree to:

1. Not execute hardening scripts on production systems or systems you do not own.
2. Always test scripts in isolated virtual machine environments first.
3. Assume full responsibility for any system changes resulting from script execution.
4. Acknowledge that these scripts implement partial CIS Benchmarks for learning and are not certified compliance tools.
